# CleanWave 系统使用手册

## 系统概述

CleanWave 是一套专为洗衣店设计的综合管理系统，旨在提高洗衣店的运营效率和客户满意度。本系统涵盖了从会员管理、订单处理到财务统计等全方位的功能，帮助洗衣店实现数字化转型和精细化管理。
